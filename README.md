# Task 3 - Login Page UI

## Project Overview

This project is a modern Login Page UI built using HTML, CSS, and JavaScript. The interface provides a clean and responsive login form with email and password fields, a login button, and a forgot password option.

## Features

* Email Input Field
* Password Input Field
* Login Button
* Forgot Password Link
* Responsive Design
* Modern User Interface
* Mobile Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

├── index.html
├── style.css
└── script.js

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Functionality

* Users can enter their email address.
* Users can enter their password.
* Login button is provided for authentication UI.
* Forgot Password link is available.
* Layout adjusts automatically on mobile and desktop devices.

## Learning Outcomes

* Form Design
* Responsive Layout
* CSS Styling
* User Interface Design
* Basic JavaScript Integration
